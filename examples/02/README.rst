==========================
Komendy wewnątrz kontenera
==========================

Uruchom kontener::

    docker run -it ubuntu:14.04 bash

Wywołaj wewnątrz niego poniższe komendy.
Sprawdź jaki jest ich rezultat i porównaj z wynikiem otrzymanym na Twoim hoście::

    ps -ef

    df -h

    ip addr list

    id
